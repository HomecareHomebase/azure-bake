### Enhancements

* renamed ingredient from `ingredient-api-managment-base`
* removed ARM template in exchange for API to enable full customization without ingredient change
* added support for the following:
  * diagnostics
  * named values
  * groups
  * users
  * subscriptions
  * products
  * loggers
  * authServers
  * identityProviders
